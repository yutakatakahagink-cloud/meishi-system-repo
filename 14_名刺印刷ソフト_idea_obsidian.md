# [[名刺印刷ソフト]]

> PC・スマホのブラウザから使う名刺の印刷・管理 Web アプリ。[[安全衛生管理システム]] や [[勤怠管理システム]] と同じ [[Firebase]]（[[GitHub Pages]] 公開）構成で、データは[[全社共通情報]]・[[部署共通情報]]・[[個人情報]]の3段階で管理する。

## ファイル構成

| ファイル | 説明 |
|---|---|
| `meishi-app/public/index.html` | アプリ本体。[[ログイン]]→[[ホーム]]→[[名刺印刷]]/[[名刺管理]]の画面遷移とUI |
| `meishi-app/public/meishi-store.js` | データ層。[[Firebase]] Realtime DB `/meishi_data` 同期＋localStorageフォールバック |
| `meishi-app/public/config.js` | [[Firebase]] 設定（[[安全衛生管理システム]]と同じ hiyarihatt-report を共用） |
| `meishi-app/start.bat` | ローカル起動（http://127.0.0.1:8791/） |
| `README.md` | 使い方・構成 |

## グラフビュー用リンクマップ

- [[名刺印刷ソフト]] → [[GitHub Pages]]（PC・携帯から閲覧）
- [[名刺印刷ソフト]] → [[Firebase]]（全端末共有・[[安全衛生管理システム]]と共用）
- [[名刺印刷]] → [[名刺プレビュー]]（91mm×55mm 実寸表示・ブラウザ印刷）
- [[名刺管理]] → [[全社共通情報]] → [[部署共通情報]] → [[個人情報]]（継承の優先順位）
- [[全社共通情報]] → [[ロゴ画像]]（部署共通で上書き可）
